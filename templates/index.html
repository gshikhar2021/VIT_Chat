VIT Chat

VIT Chat is a simple chat application that helps you connect with other people around the world.

VIT Chat is a fun side project of mine with basic features such as real-time messaging, chat rooms, authentication, and more coming soon :)

Meet new people!

Search and select a chat room to join without any lengthy sign-up forms that ask for your personal information.

Create your own chat rooms!

Can't find an interesting chat room? Just create your own! Enhance and tailor chat rooms to your needs by setting passwords, descriptions, and customizing visibility.

Chat securely with confidence!

In addition to standard TLS encryption, private and secret chat rooms require a password to enter. Access will be authorized using JSON web tokens.
